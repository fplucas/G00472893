# Final project of the Mobile Application Development subject

The project consists in an Ionic 7.2.1 + Angular application that makes use of the [Spoonacular API](https://spoonacular.com/food-api/docs).

## How to use it

1. Clone the project

```sh
git clone https://github.com/fplucas/G00472893.git
```

1. Cd the project

```sh
cd G00472893
```

1. Install dependencies

```sh
npm install
```

1. Serve web pages

```sh
npx @ionic/cli@7.2.1 serve
```
